# AutonomyOS

AutonomyOS is a lightweight, modular framework for building autonomous marketing systems.  
It gives you agents, a task queue, a scheduler, and a clean runtime so you can automate growth loops, content engines, affiliate workflows, analytics cycles, and more.

## Features
- Agent-based architecture
- Centralized runtime
- FIFO task queue
- Recurring scheduler
- Health monitoring
- Easy extensibility
